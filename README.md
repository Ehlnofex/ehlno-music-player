ehlno-music-player
==================

A web music platform. Listen to the music you uploaded.

---

This is a basic project using PHP and MongoDB for the back-end. On the
front-end there is use of JavaScript anda bit of jQuery.

This project uses:

+ Twitter Bootsrap
+ Font Awesome
+ DataTables
